# Screenshots

This directory holds the images and GIFs referenced from the repo `README.md`.

## Expected files

| Filename | Description | Recommended size |
| --- | --- | --- |
| `hero.gif` | Animated demo for the top of the README — show AI creating cards, drag-and-drop, or the timeline | 560px wide, 5–10s loop |
| `board.png` | Main Kanban board view with a few columns and cards | 1600px wide |
| `ai-assistant.png` | AI assistant panel open with a natural-language command | 1600px wide |
| `timeline.png` | Timeline / Gantt view with target dates | 1600px wide |
| `card-editor.png` | Card editor dialog showing labels, cover, checklists | 1200px wide |

## Capture tips

- Use a clean board (no personal data) — create a sample board for screenshots
- Dark mode with the default cyberpunk theme looks best
- For GIFs, tools like [Kap](https://getkap.co), [LICEcap](https://www.cockos.com/licecap/), or Cleanshot X work well
- Keep GIFs under ~4MB for fast page loads; compress with [gifski](https://gif.ski/) or [ezgif.com](https://ezgif.com/optimize)
- PNGs: run through [TinyPNG](https://tinypng.com) or `pngquant` before committing

## Updating the README

When you add or rename a file here, update the matching `<img>` / `![...]()` tags in the top-level [`README.md`](../../README.md).
